Testez PayPlug au sein de votre boutique WooCommerce pour évaluer son efficacité sans faire de paiement réel.

== Description ==

[PayPlug](https://www.payplug.fr/ "PayPlug") propose une solution de paiement très avantageuse pour les activités basées en Europe.

### Attention : ###
'Woocommerce PayPlug' vous permet de tester PayPlug au sein de votre boutique WooCommerce en utilisant le mode TEST (c'est à dire sans paiement réel).  
[Informations concernant le mode TEST de PayPlug](http://support.payplug.fr/customer/portal/articles/1701656)  
NOTE : Aucun support technique n'est assuré avec ce plugin

### Version complète :###
Pour obtenir la version du plugin permettant d'effectuer des paiements réels et d'avoir un support technique,  
[veuillez vous rendre ici](http://wba.fr/woopayplug/)

La version complète permet de :

1.  Faire des paiements réels
2.  Obtenir des mises à jour
3.  Bénéficier d'un support technique
4.  Faciliter l'installation sur les serveurs non compatibles
5.  Marquer la commande comme 'terminée' après la confirmation de paiement par PayPlug à la place de 'en cours'


== Installation ==

1. Créez [un compte payPlug gratuit](http://url.wba.fr/payplug/ "compte PayPlug gratuit")
2. Installez le plugin via le manager Wordpress ou FTP
3. Activez le plugin
4. Allez dans les paramètres WooCommerce > onglet 'Commande'
5. Cliquez sur 'PayPlug' et indiquez vos identifiants et mot de passe PayPlug

== Frequently Asked Questions ==
**Je ne vois pas d'argent sur mon compte PayPlug**  
Cette version vous permet seulement de faire des **tests de paiement** via PayPlug

**Comment puis-je faire des paiements réels**  
Vous trouverez la version complète à cette [adresse](http://wba.fr/woopayplug/)

**Une erreur se produit lors de la configuration**  
Aucun support technique n'est assuré avec ce plugin

**Comment puis-je savoir que la version complète fonctionnera si cette version ne fonctionne pas**  
Jusqu'à présent, toutes les installations ont pleinement fonctionné avec l'aide du support technique
